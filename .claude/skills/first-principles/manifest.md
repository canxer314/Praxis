# First Principles Analysis

## 核心理念

绝大多数人通过类比推理（"X 像 Y，所以 X 应该用 Y 的方法"）。第一性原理分析要求**忘掉所有已有的解决方案**，从最基本的不可否认的事实出发，重新推导。

## 适用场景

- 分析陌生领域的问题（没有可靠的类比对象）
- 挑战行业惯例或现有假设
- 创新性设计（需要跳出"大家都是这么做的"）
- 判别一个复杂主张是否成立

## 不适用场景

- 日常操作性决策（类比推理更快且足够好）
- 已有成熟最佳实践的标准问题
- 紧急决策（第一性原理分析需要时间）

## 此插件提供

- **rules.md** — 分析时必须遵守的推理规则
- **skill.md** — `/first-principles` 命令，引导用户完成结构化分析
